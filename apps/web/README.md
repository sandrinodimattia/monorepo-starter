# React Web Application

## Configuration

Create a `.env` file with the following:

```
VITE_FOO=123
```

## Getting Started

1. Install dependencies:

   ```bash
   pnpm install
   ```

2. Build all packages:

   ```bash
   pnpm run build
   ```

3. Run the CLI:

   ```bash
   # From the root directory
   pnpm --filter=@repo/web run dev
   ```
